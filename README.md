# 📓 My Journal  
> A highly customized Obsidian note vault template for journaling, productivity, and knowledge management

<img width="100%" height="auto" alt="image" src="https://github.com/user-attachments/assets/c1829076-0237-409b-bde5-e1b8d0987be3" />

Welcome to **My Journal**, a thoughtfully crafted Obsidian vault designed to help you organize your thoughts, track habits, plan your day, and grow your personal knowledge base. This vault is built on **Obsidian v1.12.7** and enhanced with a powerful suite of community plugins to create a seamless, beautiful, and functional workspace.

---

## 🌟 Features

- ✅ **Customized layout** for journaling, daily planning, and long-term knowledge tracking  
- 📅 **Integrated calendar views** for time-based organization  
- 📊 **Data-driven insights** with Dataview and Heatmap Calendar  
- 🧩 **Extensible with powerful plugins** to tailor your workflow  
- 🎨 **Beautiful visual feedback** with Live Background and File Color  
- 📝 **Efficient note-taking** with Templater and Editing Toolbar  
- 🗂️ **Enhanced file navigation** using File Explorer++ and Vertical Tabs  
- 🔄 **Kanban boards** for project and task management  
- 💡 **Pretty Properties** for clean, readable metadata  

---

## 🛠️ Plugins Used

| Plugin | Purpose |
|-------|--------|
| 🔗 **Advanced URI** | Enables advanced linking and navigation across notes |
| 📅 **Calendar** | View notes in a calendar format |
| 📊 **Dataview** | Query and display notes dynamically using SQL-like syntax |
| ✏️ **Editing Toolbar** | Quick access to formatting tools in the editor |
| 🎨 **File Color** | Color-code notes for visual organization |
| 📁 **File Explorer++** | Enhanced file browser with search and filtering |
| 📈 **Heatmap Calendar** | Visualize your activity over time |
| 📦 **Kanban** | Manage tasks and projects with Kanban boards |
| 🌈 **Live Background** | Animated, customizable background for a personalized feel |
| 📋 **Pretty Properties** | Display metadata in a clean, readable format |
| 🧩 **Templater** | Automate note creation with templates |
| 📚 **Vertical Tabs** | Organize notes in vertical tabs for better screen space use |

> *All plugins are community-developed and actively maintained. Compatible with Obsidian v1.12.7.*

## 🧩 Getting Started

1. **Download Obsidian** (v1.12.7 or later) from [obsidian.md](https://obsidian.md)
2. **Clone or download** this repository to your local machine
3. **Open the vault** in Obsidian
4. **Enable plugins** from the Community Plugins section
5. **Customize** templates, styles, and settings to match your workflow

> 💡 Tip: Use the **Templater** plugin to generate new daily notes automatically!

---

## 🎨 Customization

You can further personalize your vault by:

- Editing `css.css` for visual styling
- Modifying templates in the `Templates/` folder
- Changing file colors via the **File Color** plugin
- Setting up your own Kanban boards or calendar views
